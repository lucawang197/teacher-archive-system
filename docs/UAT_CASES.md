# UAT 验收用例（学校业务方）

## 场景 1：教师上传档案
前置条件：教师账号已启用。
步骤：
1. 教师登录系统
2. 进入档案管理
3. 选择档案类型并填写信息
4. 上传 PDF/JPG/PNG 附件
5. 提交
预期结果：
- 档案状态为“待审核”
- 教师可在个人列表中看到该记录
- 系统记录上传日志

## 场景 2：管理员审核档案
步骤：
1. 管理员登录
2. 进入档案审核列表
3. 打开待审核档案
4. 选择通过或驳回并填写意见
预期结果：
- 审核状态更新正确
- 审核人和审核时间被记录
- 教师可看到审核结果和意见

## 场景 3：附件在线预览与下载
步骤：
1. 教师或管理员打开档案详情
2. 点击附件预览
3. 点击下载
预期结果：
- 浏览器可直接预览 PDF/图片
- 下载文件名正确
- 非本人教师不可访问他人附件

## 场景 4：评分模板配置与自动评分
步骤：
1. 管理员创建评分模板
2. 录入指标、权重与评分规则
3. 启用模板
4. 执行评分
预期结果：
- 系统生成每位教师评分结果
- 展示总分、明细、排名
- 缺失项被标记

## 场景 5：Excel 导出
步骤：
1. 管理员进入数据中心或评分结果中心
2. 点击导出
预期结果：
- 成功下载 Excel 文件
- 文件可被 Office/WPS 正常打开
- 数据字段完整

## 场景 6：备份记录与恢复入口
步骤：
1. 管理员发起手动备份
2. 查看备份列表
3. 触发恢复入口
预期结果：
- 备份记录生成
- 可查询历史备份记录
- 恢复入口有日志留痕
