# 第六阶段更新说明（可上线前联调版）

本轮补强的目标是把项目从“源码与文档交付版”推进到“更接近可上线前联调版”。

## 本轮新增内容

1. 前端补齐运行骨架
   - 新增 Vite 入口、主应用、路由、认证 store、后台布局。
   - 新增登录页、工作台、备份中心、操作审计、登录日志页面。
   - 新增统一样式文件和开发/生产环境变量。

2. 部署与联调增强
   - 新增 `frontend/vite.config.js`，支持本地开发代理到后端接口。
   - 新增 `deploy/docker-compose.yml` 与 `deploy/nginx/frontend.conf`，便于 Linux + Nginx 场景落地。
   - 新增 `scripts/preflight_check.sh`，用于部署前基础检查。

3. 交付与验收材料补齐
   - 新增页面地图、接口回归检查清单、生产部署指南、学校现场 UAT 流程。

## 当前状态说明

- 该版本已经具备前端控制台运行骨架与联调结构。
- 由于当前环境无法下载 Maven 依赖，本轮仍未在沙箱内执行完整 Spring Boot 构建。
- 部分后端代码仍依赖未落盘的实体、仓储、DTO、工具类，属于下一阶段需要继续补全的范围。

## 下一阶段建议

1. 补齐后端实体、Repository、DTO、统一返回结构。
2. 补齐认证、档案、用户、考评、统计等核心 Controller/Service 全量实现。
3. 在具备 Maven 的环境中跑通 `mvn test`、`mvn package`。
4. 在一台 Linux 服务器上完成 MySQL、后端、前端的端到端联调。
