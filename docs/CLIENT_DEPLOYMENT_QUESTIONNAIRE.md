# 学校部署环境确认清单

## 1. 服务器与网络
- 应用服务器数量：
- 数据库服务器是否独立：
- 是否有单独文件存储/NAS：
- 系统是内网访问还是公网访问：
- 是否需要通过 VPN 访问：

## 2. 操作系统与中间件
- Linux 发行版与版本：
- 是否允许 Docker 部署：
- 是否已有 Nginx：
- 是否已有 MySQL 8：

## 3. 存储与备份
- 附件预计容量：
- 备份保留周期：
- 是否要求数据库与附件分开备份：
- 是否需要恢复演练：

## 4. 安全与合规
- 是否要求 HTTPS：
- 是否要求后台仅办公网访问：
- 是否要求操作日志保留时间：
- 是否要求文件上传做病毒扫描：

## 5. 对接与扩展
- 是否需要对接学校统一身份认证：
- 是否需要对接教务系统/人事系统：
- 是否需要短信或邮箱找回密码：
