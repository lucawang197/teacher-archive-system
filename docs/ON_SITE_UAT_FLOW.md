# 学校现场 UAT 流程建议

## 第一阶段：基础环境确认
1. 确认访问地址、账号、浏览器版本。
2. 确认数据库、附件目录、备份目录已配置。
3. 确认运维联系人与回滚方案。

## 第二阶段：教师端验收
1. 教师登录。
2. 修改个人信息。
3. 上传一份论文档案并附带 PDF。
4. 查看档案状态。
5. 查看个人数据与考评结果。

## 第三阶段：管理员端验收
1. 管理员登录。
2. 在档案管理中审核教师材料。
3. 在数据中心筛选并导出数据。
4. 新建或启用评分模板并执行评分。
5. 查看评分结果与备注。
6. 新增教师用户。

## 第四阶段：运维与审计验收
1. 查看操作审计日志。
2. 查看登录日志。
3. 执行手动备份。
4. 查看健康检查接口返回。

## 第五阶段：问题记录与结论
1. 记录缺陷与建议。
2. 明确上线前必须修复项。
3. 确认试运行范围和时间。
