# 接口回归检查清单

## 1. 认证与权限
- 登录成功：管理员、教师账号均可正常登录。
- 登录失败：密码错误、账号禁用、超限锁定返回正确提示。
- Token 过期：前端应自动跳转至登录页。
- 权限隔离：教师不可访问管理员接口。

## 2. 档案管理
- 教师新增档案并上传附件。
- 教师编辑未审核档案。
- 教师删除未审核档案。
- 管理员查看全部档案并按状态/教师筛选。
- 管理员审核通过、审核驳回。
- 附件预览、附件下载。

## 3. 数据中心
- 管理员查看全校汇总数据。
- 教师查看个人数据。
- 汇总导出 Excel 成功。

## 4. 考评管理
- 新建评分模板。
- 模板生效与停用。
- 执行自动评分。
- 查看评分结果与排名。
- 评分结果备注。
- 考评结果导出。

## 5. 用户管理
- 新增教师账号。
- 编辑教师信息。
- 启用/禁用账号。
- Excel 批量导入。

## 6. 日志与备份
- 操作审计查询。
- 登录日志查询。
- 审计日志导出。
- 手动备份。
- 备份记录查询。
- 恢复请求留痕。

## 7. 运维与健康检查
- `/health` 返回服务状态。
- 生产环境配置加载正确。
- 备份脚本、恢复脚本、部署脚本可执行。
