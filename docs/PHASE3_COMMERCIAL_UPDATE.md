# 第三阶段增强（商用交付补强）

本轮补强聚焦“学校正式上线可交付”的要求，而不是仅满足课程演示。

## 本轮新增内容

### 1. 附件在线预览与下载接口
新增接口：
- `GET /api/attachments/{id}/preview`
- `GET /api/attachments/{id}/download`

补强点：
- 管理员可访问全部附件
- 教师仅可访问本人附件
- 已删除档案对应附件不可访问
- 响应头支持中文文件名下载

### 2. 备份记录分页查询
新增接口：
- `GET /api/backups?page=0&size=10`

用途：
- 支持学校管理员查看历史备份记录
- 便于验收时展示备份留痕与恢复入口

### 3. 生产环境配置样例
新增文件：
- `backend/src/main/resources/application-prod.yml`
- `deploy/.env.example`

用途：
- 明确生产环境用环境变量覆盖数据库、JWT 密钥、文件存储目录
- 降低交付后部署沟通成本

### 4. 自动化冒烟脚本
新增文件：
- `scripts/api_smoke_test.sh`

作用：
- 登录
- 校验工作台摘要接口
- 校验教师列表接口
- 校验档案汇总接口
- 校验评分模板接口

适合用于：
- 部署后快速回归
- 学校现场演示前自查
- 每次更新后做基础健康检查

## 建议作为上线前必做项
- 接入 HTTPS 证书
- 将附件目录挂载到独立磁盘或 NAS
- 将数据库备份改为 `mysqldump` + 定时任务
- 增加杀毒扫描或网关级文件安全检查
- 将操作日志接入学校现有运维监控平台
