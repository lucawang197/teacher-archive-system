# 前端页面地图

## 公共页
- 登录页 `/login`

## 管理员与教师共用
- 工作台 `/dashboard`
- 档案管理 `/archives`
- 数据中心 `/data`
- 考评管理 `/evaluation`

## 仅管理员
- 用户管理 `/users`
- 操作审计 `/logs`
- 登录日志 `/login-audits`
- 运维中心 `/ops`
- 备份中心 `/backups`

## 页面职责说明
- 工作台：展示系统概况、待办和快捷入口。
- 档案管理：上传、编辑、审核、预览附件。
- 数据中心：按教师/学科汇总，支持导出。
- 考评管理：模板维护、执行评分、查看结果。
- 用户管理：教师账号维护。
- 操作审计：关键操作记录。
- 登录日志：登录行为留痕。
- 运维中心：健康检查、备份概览、部署建议。
- 备份中心：手动备份与恢复申请。
